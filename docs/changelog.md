# 更新日志

- 增加插件模块，包含内置的插件编辑器，支持用户自定义插件
- 优化卡片窗口的拖动功能，可自动按图标行列数目布局
- 优化UI界面

## 1.0.4 (2025-05-08)
- 新增软件更新的功能，可以在软件中直接升级新版本了
- 将图表右键菜单替换为系统的右键菜单，支持更多功能

- 取消拖动时窗口半透明效果，便于实时观察窗口动态

## 1.0.2 (2025-04-30)
